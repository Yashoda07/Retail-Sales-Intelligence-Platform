# Data Dictionary

| Column | Data Type | Description |
|---------|-----------|-------------|
| row_id | Integer | Unique identifier for each row |
| order_id | String | Unique identifier for each order |
| order_date | Date | Date on which the order was placed |
| ship_date | Date | Date on which the order was shipped |
| ship_mode | String | Shipping method used for the order |
| customer_id | String | Unique identifier for each customer |
| customer_name | String | Name of the customer |
| segment | String | Customer segment (Consumer, Corporate, Home Office) |
| city | String | City where the order was placed |
| state | String | State where the order was placed |
| country | String | Country where the order was placed |
| region | String | Geographic region of the order |
| market | String | Market in which the order belongs |
| market2 | String | Additional market classification |
| product_id | String | Unique identifier for each product |
| category | String | Main product category |
| sub_category | String | Product sub-category |
| product_name | String | Name of the product |
| sales | Float | Total sales amount of the order |
| quantity | Integer | Number of units sold |
| discount | Float | Discount applied to the order |
| profit | Float | Profit earned from the order |
| shipping_cost | Float | Cost of shipping the order |
| order_priority | String | Priority level assigned to the order |
| year | Integer | Year of the order |
| weeknum | Integer | Week number of the order |
| ji_lu_shu | Unknown | Unknown column; to be investigated during data cleaning |
